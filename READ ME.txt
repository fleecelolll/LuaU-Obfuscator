FLEECE LUA OBFUSCATOR

INSTALL

1. Extract the whole folder from the ZIP. Do not run it from inside the ZIP.
2. Double-click Installer.bat and leave the window open until every check passes.

You do not need to run the installer as administrator.

USE

Double-click Lua Obfuscator.pyw, choose a .lua or .luau file, select Lua 5.4 or Roblox Luau, select an obfuscation level, choose the output folder, and click Obfuscate.

Lua output is saved as <name>.obfuscated.lua. Luau output is saved as <name>.obfuscated.luau. The original file is never overwritten.

TARGETS

- Lua 5.4 uses every protection available in Hercules.
- Roblox Luau automatically skips the VM and bytecode protections that are incompatible with Luau.

UPDATES

Run Installer.bat again to repair Hercules, Lua 5.4, the private Python environment, or PySide6.

WHAT THE INSTALLER ADDS

- A private .venv folder keeps PySide6 separate from globally installed Python packages.
- If compatible 64-bit Python 3.10 through 3.14 is unavailable, setup installs Python 3.13 for the current Windows user through winget.
- A pinned Hercules source snapshot is stored inside .runtime.
- Lua 5.4.8 from LuaBinaries is stored inside .runtime.
- The Hercules and Lua archives are verified against pinned SHA-256 checksums.

PRIVACY

The app contains no telemetry, analytics, accounts, or usage tracking. Scripts are processed locally and are never uploaded. Installer.bat connects only to the official setup sources described above.

TROUBLESHOOTING

If setup fails, check your internet connection and run Installer.bat again. Details are saved in setup.log. The log can include local folder paths, so review it before sharing it.

If Lua Obfuscator.pyw does not open, rerun Installer.bat and make sure every final check passes. Keep Installer.bat and Lua Obfuscator.pyw together in the same folder.

REMOVE

Close the app and delete this folder. That removes the private Python environment, Hercules, Lua 5.4, and the app files. A Python installation added through winget may remain in your user environment.
