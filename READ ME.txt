FLEECE LUAU OBFUSCATOR

INSTALL

1. Extract the whole folder from the ZIP. Do not run it from inside the ZIP.
2. Double-click Installer.bat and leave the window open until every check passes.

You do not need to run the installer as administrator.

USE

Double-click LuaU Obfuscator.pyw, choose a .lua file, select an obfuscation level, choose the output folder, and click Obfuscate.

UPDATES

Run Installer.bat again to repair Prometheus, Lua 5.1, Python, or PySide6.

WHAT THE INSTALLER ADDS

- If Python is already installed, setup uses it and installs or updates PySide6 in that Python environment.
- If Python is unavailable, setup installs Python 3.13 for the current Windows user through winget.
- Prometheus comes from the official prometheus-lua/Prometheus GitHub repository and stays inside the app folder.
- Lua 5.1.5 comes from LuaBinaries and stays inside the app folder.

PRIVACY

The app contains no telemetry, analytics, accounts, or usage tracking. Installer.bat connects to the official setup sources described above to install or repair components.

TROUBLESHOOTING

If setup fails, check your internet connection and run Installer.bat again.

If LuaU Obfuscator.pyw does not open, rerun Installer.bat and make sure every final check passes. Keep the whole folder together; moving only the .pyw file will break the app.

REMOVE

Close the app and delete this folder to remove Prometheus, Lua 5.1, and the app files. Python or PySide6 installed by setup may remain in your user environment.
